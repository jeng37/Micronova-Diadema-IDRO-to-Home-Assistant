# SyncTERM bOX builds

This folder contains the preconfigured SyncTERM package for connecting to the live bOX PCBoard BBS.

Package:

```text
syncterm_v1.9.rc3_bOX_ALL_BUILDS.zip
```

Connection:

```text
Host:     mybbs.duckdns.org
Port:     2222
Protocol: SSH
Login:    bbs
Password: bbs
```

Quick start:

```bash
ssh bbs@mybbs.duckdns.org -p 2222
```
